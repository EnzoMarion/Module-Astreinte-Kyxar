Docker -> plateforme conçue pour aider les développeurs à créer, partager et exécuter des applications modernes. Nous nous occupons de la configuration fastidieuse, afin que vous puissiez vous concentrer sur le code.

Unbuntu -> système d'exploitation GNU/Linux basé sur la distribution Debian. Il est libre, gratuit, et simple d'utilisation.

PhpStorm -> Outil de développement pour les projets web et PHP. C'est l'environnement de développement idéal pour travailler.

Redmine -> Application web flexible de gestion de projet. Écrit en utilisant le framework Ruby on Rails.

Github -> plate-forme d'hébergement de code pour le contrôle des versions et la collaboration.

GitLab -> logiciel libre de forge basé sur git proposant les fonctionnalités de wiki, un système de suivi des bugs et une gestion de projet d'équipe.

GitBash -> Application pour les environnements Microsoft Windows qui fournit une couche d'émulation pour une expérience de ligne de commande Git.
